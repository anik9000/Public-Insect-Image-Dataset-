# Public-Insect-Image-Dataset-
The dataset contains a total of 1,774 images with 1920 Ã— 1280 pixels each, taken by CANON EOS 760D P - DSLR Camera to capture 24 different classes of insects. The images are captured from nearby agricultural lands from paddy, maize and sugarcane fields near Tanjore district, Tamil Nadu, India. The insect samples are captured with different poses and all the insect images are in lossless 24-bit-color JPG format. The insects are:
